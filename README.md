# Cheyserr Bayan-Alejado — Portfolio

A single-page portfolio site for Cheyserr Bayan-Alejado, a virtual assistant specializing in admin operations, CRM, and lead generation for healthcare and service-based businesses.

## Structure

- `index.html` — the entire site (HTML, CSS, and JS in one self-contained file)
- `transparent-blooming.png` — hero portrait
- `Cheyserr-Bayan-Alejado-Resume.pdf` — downloadable résumé linked from the site

## Running locally

It's a static page, so any static server works. For example:

```bash
python -m http.server 4173
```

Then open http://localhost:4173.

## Tech

Plain HTML, CSS, and vanilla JavaScript. Fonts (Fraunces and Inter) load from Google Fonts. No build step and no dependencies.
